# Assembly Guide

Status Beacon v0.1 Alpha

1. Print the enclosure.
2. Print or cut the frosted diffuser.
3. Mount the LED board behind the diffuser.
4. Mount the ESP32-C3.
5. Mount the accelerometer flat inside the case.
6. Connect the USB-C charging module.
7. Install the LiPo battery.
8. Install the power switch.
9. Upload BeaconOS firmware.
10. Test green, red, yellow, and blue status modes.
