# STL Export Guide

## OpenSCAD Render

Use **F6**, not F5, before exporting.

- F5 = preview only
- F6 = final render for STL export

## Export Folder

Save exported STL files here:

`CAD/STL/LightCube/`

## Version Naming

Use this naming style:

`StatusBeacon_LightCube_<PartName>_v0.1.stl`

Example:

`StatusBeacon_LightCube_Body_v0.1.stl`

## Print Notes

The body is a prototype and may need small changes depending on the exact ESP32, LED ring, charger module, and battery you buy.
